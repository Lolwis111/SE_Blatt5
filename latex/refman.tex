    \section{How to complile}
        To compile the library-API you have to go to the source directory and run:\\

        \textit{make}\\

        If you want to clean the source directory (delete old object files and the binary)
        you can run:\\

        \textit{make clean}\\
    
        If you want to rebuild this documentation, you can run:\\

        \textit{make doc}\\

        (This requires doxygen and latex to be installed)
